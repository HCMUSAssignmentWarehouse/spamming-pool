My JARS: A program that let user enter income in a month, then divides this income into 6 parts acording to JARS plan

	Youtube Demo: https://youtu.be/7PgavuMDwxY

	Main flow:
	User insert 1000 to amount of money edit control and press Calculate button: Each edit control show the money: 
	Essential spending: 550, Investment: 100, Education: 100, Savings: 100.
	After that A Dialog show and ask whether user agree to give away 5% of his/her money
		+ User agree, press "Of course!": Enjoyment money show 100, Give away money appear and show: 50
		+ User disagree, press "Next time!": Enjoyment money show 150

	Additional flow:
		- User insert character or symbol to amount of money edit control, and window dont regconise this input, and show a warning message.
		- User left the amount of money edit control empty, press Calculate button, an error Message Box show "Amount of Money must not be empty!"
